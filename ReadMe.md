# WaniKani Vocab Practice
Practice vocab from WaniKani lessons. Uses sentence examples from WaniKani to practice with. Allows selecting from lessons in a range, subjects available by a date, or critical vocab (accuracy < 75%).

Must provide API Token from WaniKani to use. This can be retreived/generated here: https://www.wanikani.com/settings/personal_access_tokens
